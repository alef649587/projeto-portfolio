Projeto Dois

Este projeto se concentra em um aplicativo móvel para monitorar metas de condicionamento físico e saúde.

Ver no Github

Detalhes do Projeto

Este projeto usa React Native para construir um aplicativo móvel multiplataforma que rastreia as atividades de condicionamento físico e métricas de saúde do usuário. Inclui recursos como registro de treino, acompanhamento de progresso e integração com APIs externas.
